# ⚖️ Round Steel Pipe Weight Calculator

**Approximate weight calculator for round steel pipe**  
Calculate the weight per linear meter and the total weight for round carbon steel or stainless steel pipes.

---

## 📋 Description

This tool provides a fast and reliable way to estimate the weight of round steel pipes based on standard dimensions and material density.

Whether you're working with **carbon steel** or **stainless steel**, this calculator helps engineers, fabricators, and procurement teams quickly determine:

- ⚙️ **Weight per linear meter** (kg/m)
- 📏 **Total weight** for any given pipe length (kg)

---

## 🔩 Supported Materials

| Material | Density |
|---|---|
| Carbon Steel | 7.85 g/cm³ |
| Stainless Steel | 7.98 g/cm³ |

---

## 📐 Formula Used

The weight per linear meter is calculated using the standard pipe weight formula:

```
W = π × (D - t) × t × ρ
```

Where:
- `W` = weight per meter (kg/m)
- `D` = outer diameter (mm)
- `t` = wall thickness (mm)
- `ρ` = material density (g/cm³)

---

## 🚀 How to Use

1. Enter the **outer diameter** (mm)
2. Enter the **wall thickness** (mm)
3. Enter the **pipe length** (m)
4. Select the **material type** (carbon steel / stainless steel)
5. Get instant results for weight/meter and total weight
